Learning more about programming audio in general

  • Let's Learn About Waveforms, and interactive tutorial by Josh Comeau.
  • Coursera: Audio Signal Processing for Music Applicaitons This course uses Python and gets you through all of the Fourier transform stuff, different methods of modelling audio synthesis, and into advanced techniques like time stretching. It is very hands-on and requires you to write all of the code to make the assignments work.
  • Book: BasicSynth by Daniel R. Mitchell. This is a good intro to audio programming with C/C++.
  • Books: by Will Pirkle. You are well into C/C++ at this point. These are textbooks and are exhaustive. I haven't read the 2nd editions, but they look to be much updated and I look forward to getting and reading them.
